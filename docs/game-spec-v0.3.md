# Browser Idle RPG Ver0.3

## コンセプト

半放置型ブラウザRPG

プレイヤーは複数の冒険者パーティを編成し、
自動で冒険を進める。

## キャラクター

- 最大20人所持
- パーティ最大6人
- 複数パーティ運用可能

## 装備

- 武器
- サブ装備
- 防具

計3スロット

## スキル

- 最大3個（将来8個予定）

## ジョブ

- ジョブごとに成長傾向
- レベルアップ時に多少ランダム成長

## ダンジョン

初期は1箇所

試作時間：
3分

## モンスター

初期3種類

## 図鑑

Lv0～Lv5

### Lv1

- 名前
- 生息地

### Lv2

- スキル
- ドロップ1
- ドロップ2

### Lv3

- ドロップ3
- ドロップ4

### Lv4

- 弱点
- 耐性

### Lv5

- レア情報

## AI

### 性格

キャラクターごとの個性

例

- 慎重
- 勇敢
- 冷静
- 臆病

### 作戦

例

- バランス
- 攻撃重視
- 回復重視
- MP節約

### 図鑑レベル

図鑑が育つと判断精度向上

## 冒険ログ

帰還時に詳細表示

## 保存

- LocalStorage
- 将来的にクラウド保存対応
